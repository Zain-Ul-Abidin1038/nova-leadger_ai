## 🚀 Getting Started

### First Launch
1. **Sign Up** - Create account with email/password
2. **Grant Permissions** - Allow camera, storage, location access
3. **Add API Key** - Configure Nova API key in settings
4. **Scan First Receipt** - Take photo of a receipt to test
5. **Explore Features** - Try chat, voice input, analytics

### Configuration
1. Open Settings
2. Navigate to API Configuration
3. Enter your Nova API key
4. (Optional) Configure AWS Amplify credentials
5. Save and restart app

### Getting Nova API Key
1. Visit [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Sign in with Google account
3. Create new API key
4. Copy and paste into app settings
