## 🚀 Getting Started

### Prerequisites
- Flutter SDK 3.10.8+
- AWS Account with Bedrock access
- AWS Amplify CLI

### Installation

```bash
# Clone repository
git clone https://github.com/Zain-Ul-Abidin1038/nova-finance-os.git
cd nova-finance-os

# Install dependencies
flutter pub get

# Configure environment
cp .env.example .env
# Edit .env with your AWS credentials

# Initialize Amplify
amplify init
amplify add auth
amplify add storage
amplify push

# Run app
flutter run
```
