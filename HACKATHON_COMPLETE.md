---

## 🌟 The Problem We're Solving

**1.7 billion people worldwide are unbanked** - they lack access to basic financial services, accounting knowledge, and tools to manage their money effectively. This creates a cycle of poverty and financial exclusion.

NovaLedger AI democratizes financial expertise by putting a **chartered accountant in everyone's pocket**, powered by AI.

---

## 💡 Our Solution

An AI-powered mobile app that provides:

1. **Intelligent Receipt Analysis** - Snap a photo, get instant tax deduction calculations
2. **Conversational Finance Management** - Chat with AI to track income, expenses, and loans
3. **Vision Nova** - Live camera-based financial advisor (like having a CA look over your shoulder)
4. **NovaNavigator** - Autonomous AI agent that can book flights, order food, and manage finances
5. **Multi-modal Input** - Voice, camera, text, location - accessible for all literacy levels

---

## 🚀 Key Features Implemented

### 1. Smart Receipt Scanning
- Camera-based receipt capture
- AI-powered OCR and analysis using Nova Vision
- Automatic tax deduction calculation (meals 50%, office 100%, alcohol 0%)
- Real-time Nova Trace showing AI reasoning

### 2. Intelligent Chat Interface
- Natural language financial queries
- Multi-transaction parsing ("I spent ₹500 on groceries and ₹200 on fuel")
- Structured finance parsing with JSON extraction
- Voice input support for accessibility

### 3. Vision Nova (Live AI Advisor)
- Real-time camera analysis every 3 seconds
- Live advice overlay on camera feed
- Instant purchase decision guidance
- Detailed analysis on capture
- Perfect for illiterate users - just point and get spoken advice

### 4. NovaNavigator (Autonomous Agent)
- AI agent that can navigate apps like a human
- Book flights, order food, pay bills autonomously
- Screen analysis and UI element detection
- Automatic financial tracking of all transactions
- Quick task templates for common actions

### 5. Comprehensive Finance Management
- Income tracking with categories
- Expense tracking with deduction calculations
- Loan/Ledger management (receivables & payables)
- Real-time balance updates
- Analytics and insights

### 6. Production-Ready DevOps
- GitHub Actions CI/CD pipeline
- Automated testing and security scanning
- Firebase hosting for web deployment
- Android APK build automation
- Multi-environment support (dev/staging/prod)

## 📱 Installation & Testing

### Prerequisites
- Flutter SDK ^3.10.8
- Firebase account (for Nova AI)
- Android Studio / Xcode (for mobile)

### Quick Start
```bash
# Clone repository
git clone https://github.com/yourusername/nova_ledger_ai.git
cd nova_ledger_ai

# Install dependencies
flutter pub get

# Configure Firebase
flutterfire configure

# Run app
flutter run
```

3. "I lent ₹10,000 to Raj and he paid back ₹3,000"
